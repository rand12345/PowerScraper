# PowerScraper
A scraper for power devices to feed data to OpenEnergyMonitor

# Supported Hardware
- Solax SK-SU5000E Wifi interface
- Solax SK-SU5000E Modbus/TCP Interface
- Eastron SDM630V2 3 Phase energy meter
